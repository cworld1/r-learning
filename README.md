## 项目介绍

根目录中的 z_Homework 为学习时的作业源代码。此外其他为 R 相关的各个部分。每个部分都有 Introduction 来简述本部分内容并做简要的引入。

不是所有章节都采用 markdown 或完整可运行的 r 文件。知识点更多会在代码或描述本身。
